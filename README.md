# Spots

Spots is an interactive image-sharing application where users can share photos, like images from other users, and make small updates to their profile. The project focuses on building a clean, responsive interface that adapts seamlessly across desktop, tablet, and mobile screen sizes.

Spots allows users to explore and interact with shared images in a simple, visually engaging layout. The main features include:

- Viewing a gallery of image cards
- Liking photos shared by other users
- Adding and removing photos
- Updating basic profile information
- Responsive layout for desktop, tablet, and mobile devices

The application is designed to prioritize usability, responsiveness, and clean UI structure while following modern frontend best practices.
